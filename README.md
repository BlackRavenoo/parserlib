# ParserLib

Приложение для скачивания контента(манги, новелл) из поддерживаемых источников с экспортом в различные форматы (pdf, epub, fb2).

## Доступные интерфейсы:
- CLI

## Поддерживаемые источники:
- MangaLib
- RanobeLib

## Поддерживаемые форматы:
- pdf
- epub
- fb2

### Установка

**С помощью uv (рекомендуется):**
```sh
uv sync
```

**С помощью pip:**
```sh
pip install -e .
```

Для CLI-интерфейса:
```sh
uv sync --extra cli
```

### Первый запуск
```sh
uv run cli.py fetch <ссылка>
```

Доступен выбор формата:
```sh
uv run cli.py fetch <ссылка> --format <формат>
# Пример
uv run cli.py fetch <ссылка> --format fb2
```

### Обновление уже скачанной книги
Режим `append` сейчас поддерживается только для `epub`.

Как пользоваться:
```sh
uv run cli.py append <ссылка> <существующий_файл>
```
