# ParserLib

Приложение для скачивания с электронных ресурсов манги, новелл в форматах pdf, epub, fb2.

Доступные интерфейсы:
- CLI

Поддерживаемые сайты:
- MangaLib

Поддерживаемые форматы:
- pdf
